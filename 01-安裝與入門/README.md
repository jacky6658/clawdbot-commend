# 01-安裝與入門

本資料夾收錄 **OpenClaw / ClawdBot 安裝與第一次使用** 相關文檔與腳本。

## 文檔一覽

| 檔案 | 說明 |
|------|------|
| **OpenClaw-全方位實作與安全指南.md** | 專案背景、安全第一、VPS/本機環境、安裝流程、引導精靈、配對、應用層安全、技能安全、疑難排解、進階技巧 |
| **OpenClaw-本機安裝與安全重裝指南.md** | 本機 Node.js 與一鍵安裝、安全備份與重新安裝五步驟 |
| **OpenClaw-安全與安裝健檢報告.md** | 本機防護對照、Warn：3、allowFrom、Control UI、憑證、技能安全 |
| **Clawdbot-新電腦還原與安裝指南.md** | 新電腦上安裝並還原舊機備份（含技能、記憶、設定） |
| **本機還原指令與安裝位置.md** | 本機還原時常用指令與安裝路徑整理 |
| **執行還原.sh** | 還原用腳本，依實際路徑與備份來源調整後使用 |

## 教學順序建議

1. 先讀「全方位實作與安全指南」建立安全觀念與整體流程。  
2. 再依「本機安裝與安全重裝指南」或「新電腦還原與安裝指南」執行安裝或還原。  
3. 安裝完成後對照「安全與安裝健檢報告」做一次防護檢查。

若另有 **OpenClaw-整合安裝教學-本機與Zeabur-人人可懂.md**（本機 + Zeabur、含無環境從零教學），可一併放在本資料夾，作為非工程師友善的整合版教學。
