# OpenClaw / ClawdBot 備份與教學指南

本倉庫整理 OpenClaw（原 ClawdBot）的**安裝、還原、問題排除與進階教學**文檔，適合放在 GitHub 供查閱，或作為**課程教材與自學**材料。文檔檔名已統一為連字號、無空格與括號，方便課程平台與學員使用。

## 建議使用方式

- **第一次安裝**：依序閱讀 `01-安裝與入門` 中的「全方位實作與安全指南」「本機安裝與安全重裝指南」，再視需要參考「新電腦還原與安裝指南」。
- **已有備份要還原**：看 `02-還原與重裝`，依自己的備份選擇對應的還原指南。
- **遇到錯誤或異常**：到 `03-問題排除與修復` 對照症狀（Gateway、記憶、模型、Antigravity 等）。
- **教學用**：可依「01 → 02 → 03 → 04」順序作為課程章節，或依學員需求單獨挑選資料夾。

---

## 上傳至 GitHub 時請注意

- **05-備份資料** 內含設定、憑證、對話與媒體等敏感內容，請勿推送到公開 repo。  
  建議在倉庫根目錄使用 `.gitignore` 排除該資料夾（見下方）。

## .gitignore 建議

若將本目錄作為 Git 倉庫，建議在根目錄加入：

```
# 備份資料含敏感資訊，不納入版本控制
05-備份資料/
.DS_Store
```
