## 使用方式

依遇到的現象選擇對應文檔：Gateway 無法啟動→埠佔用；記憶或搜尋異常→記憶功能修復；模型或 API 錯誤→穩定性與模型配置；OAuth／Antigravity→Antigravity 修復；多人或 session 混用→多人對話與大腦隔離。
